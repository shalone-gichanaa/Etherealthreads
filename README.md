## How to Use
1. Clone or download the repository.  
2. Open `index.html` in your browser.  
3. Browse **Products** to see the collection.  
4. Navigate to **Add Product** to add a new item:  
   - Enter **Product Name**  
   - Select **Category**  
   - Enter **Price**  
   - Upload **Image**  
   - Click **Add Product**  
5. Check the **Products Page** to view the newly added item.  
6. Click the **Delete** button to remove any product.  


## Known Issues
- Image uploads are limited to one per product.  
- Products stored in **localStorage** are only saved on the same browser/device.  
- Delete functionality may require refreshing in some cases.
